The objective of the task is to navigate a field of obstacles and reach the finish line. When the simulation starts, a countdown timer of 180 seconds would automatically start. It will stop when you reach the finish line, or 0 to zero if you collide with an obstacle.
The following instructions need to be followed while making a final submission video of Task 2: 1. Make sure you've followed the instructions on the Task 2 Problem Statement Page and have edited the teaminfo.json file by replacing the placeholder with your team ID.
2. Go to the metrics tab.
3. Start the simulation using the Webots interface.
4. The countdown timer will start counting down from 180 seconds.
5. As you traverse towards the goal line the completion percentage will increase, it is indicated in the Progress field.
6. If you collide with an obstacle countdown timer will go to 0.
7. Login to the eYSRC portal and upload the zip file obtained in the Task 2 folder.
