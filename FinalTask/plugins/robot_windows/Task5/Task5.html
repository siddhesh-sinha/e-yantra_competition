Time has come to combine all of the things we have learnt so far and help our intrepid explorer in it's final adventure! In task 3 we designed a wall follower using distance sensor and in Task 5 we designed a line follower using camera, and now in the final task we will be combining both these tasks. The addition in the task is to be able to switch from one controller to another. Remember to write well documented and robust code because your code carries marks in this task.
The aim of the task is to reach the blue square by collecting all the waypoints on its way. The simulation would stop when:
1. All waypoints are collected, hence completing both the wall following and line following sections or
2. 180 seconds have elapsed after starting the run.
